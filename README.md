# Vyefit

Vyefit is a focused iOS training companion for lifters who want fast workout planning, clean set/rep/weight logging, and local-first history.

## Highlights

- Create and manage custom workouts
- Organize workouts with multiple exercises
- Open a workout directly into record tracking (no start/end workout flow)
- Add records with reps + weight and edit them later
- Store both kg and lb values for each record
- See recent record history per exercise
- Build weekly schedules with workout, run, and rest slots
- Import completed workouts from Apple Health into session history

## Product Direction

Vyefit is intentionally optimized for **non-realtime training workflows**:

## Screens

- `Today`: daily summary and focus card
- `Train`: workouts and exercise record tracking
- `Schedule`: weekly planning (including run selector)
- `You`: profile, milestones, and session history
